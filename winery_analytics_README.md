
# Winery Business Analytics Platform

A Streamlit dashboard for wineries to analyze tasting room sales, wine club retention, event ROI, inventory, forecasting, and AI-style business insights.

## Features

- Executive KPI dashboard
- Sales analytics by wine and channel
- Wine club churn risk scoring
- Event profitability and ROI
- Inventory health dashboard
- Revenue forecasting
- AI-style business recommendations
- CSV templates included inside the app

## How to Run

```bash
pip install streamlit pandas numpy plotly scikit-learn
streamlit run app.py
```

## Deploy on Streamlit Cloud

1. Create a GitHub repo.
2. Upload `app.py`.
3. Go to Streamlit Cloud.
4. Connect your GitHub repo.
5. Set main file path to `app.py`.
6. Deploy.
